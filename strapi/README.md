# Strapi application

First of all, install packages

```
yarn install
```

# Available commands in your project:

```
yarn develop
```
  Start Strapi in watch mode.

```
yarn start
```
  Start Strapi without watch mode.

```
yarn build
```
  Build Strapi admin panel.

```
yarn strapi
```
  Display all available commands.

You can start by doing:

```
yarn develop
```
